# US Violent Crime Rate Analysis

## Project Overview

This Project analyzes violent crime trends across US states and counties using public health and crime-related datasets. The analysis focuses on identifying regional crime patterns, temporal changes, and high-risk areas through data analytics and visualization techniques.

## Project Objectives

* Analyze violent crime patterns across US states and counties
* Compare crime rates between regions
* Explore long-term national crime trends
* Detect counties with unusually high crime activity
* Visualize geographic and temporal crime distributions

## Core Business Questions

1. WHich US states have the highest violent crime rates?
2. Identify high-risk states and counties

## Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn

## Visualizations 
* State-wise crime ranking bar charts

## Key Insights

* Certain states consistently report crime rates above the national average. 
* Violent crime trends vary significantly across regions
