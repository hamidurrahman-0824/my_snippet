# Coffee Shop Retail Analysis

My first data analysis project using Python.

## What this project does
Analyzes coffee shop transaction data to find business insights like best-selling products, peak hours, and revenue trends.

## Libraries used
- pandas
- matplotlib
- seaborn
- numpy

## How to run
1. Put `coffee_shop.csv` inside the `DataSets/` folder
2. Open `coffeShop_retail.ipynb` in Jupyter Notebook and run all cells
